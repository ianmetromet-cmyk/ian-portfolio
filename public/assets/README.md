# Portfolio assets

Place media files here so paths like `/assets/images/...` resolve correctly in production.

Expected folders:

- `images/` — photos, logos, thumbnails
- `video/` — MP4, MOV, WAV files referenced in the site
- `docs/` — PDFs (resume, press clips, op-eds)

Copy your asset bundle into this directory before deploying. The site will still build without them, but images and videos will not appear until files are added.
